Selecting and Components

7.1 Free or Mapped Mesh

  • use the MSHKEY command to choose a free or a mapped mesh

    7.2 Setting Element Attributes

  • before meshing, you need to define the appropriate element attributs
    • the element type
    • real constant set
    • material properties
    • element coordinate system

      7.2.1 Creating Tables of Element Attributes

  • you can assign the element attributes to the solid model entities(allow you to preassign attributes for each region of the model)
  • control thickness

    7.3 Mesh Controls

  • You can establish such factors as the element shape,midside node placement,and element size to be used in meshing the solid model

    7.3.1 The MeshTool

  • MeshTool (Main Menu > Preprocessor > Meshing > MeshTool)
    • it remains open until you either close it or you exit PREP7

      7.3.2 Element shape

  • specify the element shapes you want to use:
    • area: normally triangular or quadrilateral
    • volume: hexahedral(six faces) or tetrahedral(four faces)
      7.3.2.1 A Note About Degenerate Element Shapes
  • MSHAPE: choose the 3d or 2d with what kind of shape
  • MSHKEY: free mesh or restricted mesh
