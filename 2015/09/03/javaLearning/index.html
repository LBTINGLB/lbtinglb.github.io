Sep 10th Reference Types

  • Define a class:
  • Use the new operator to instantiate the class
  • Constructor
    • If constructors are provided, you must use one
    • If no constructors are provided, Java generates a “defalut” constructor - with no arguments
  • Access Modifiers
  • public : everyone can access it
  • private : Only this class(file) can access it
    • private methods: not to expose the internal implementation details of our class
  • subclass’s constructor
    • The first line of a subclass’s constructor must be a call to the superclass constructor, or you would have an error (slides P 25)
  • toString() methods
    • you can write your own tostring() method which would overrides the superclass tostring()
  • casting:
    • convert from one type of reference to another
    • no explicit cast when going up the class hierarchy
      • from shape to object
  • what’s the difference between a primitive type and a reference type
    • Primitive type: use fixed storage
    • Reference type: use varied amounts of storage
