Notes when Learning Latex

Latex

LaTeX新人教程，一小时从完全陌生到基本入门

  • your first document

your first document

  • add title and name:

  • comment: just like matlab. use %

  • new paragraph and chapter

    • \section or add sub to make small title.
    • use \paragraph to specify a new paragraph
  • add content
    • \tableofcontents
  • an empty line means starting a new paragraph
  • \\ means forcing a line break
  • add equations

  • $…$是开启行内数学模式，用于和文本合在一起使用。

  • $$…$$和[…]是另起一行居中开启数学模式。通常用起来差别不是很大，不过$$会修改默认的公式行间距，有时可能会对文章的整体效果有影响。
  • you can google latex math for help.

  • add picture

  • insert picture in latex

  • create a simple table
    • \hline: change the display of the line
    • |c|c|: change the position of the object in the table
