    • TetGen Study Note

      1. Introduction

  • Generate tetrahedral meshes of any 3D polyhedral domains.
  • Delaunay trangulations: maximize the minimum angle of all the angles of the triangles in the triangulation
  • Voronoi Diagram: partitioning of a plane into regions based on distance to points in a specific subset of the plane
  • generate good quality and adaptive tetrahedral heshes suitable for numerical methods, as FEA and finite volume methods.

    2. Feature

  • Mesh quality, mesh size control
  • ‘-q’, ‘-q1.4’, and ‘-q1.1’, respectively. The number after ‘-q’ is a parameter - the radius-edge ratio of tetrahedron, default is 2.0. The smaller the radius-edge ratio is, the better mesh quality will achieve. The sizes of the meshes are well-graded with respect to the input boundary.

    Getting Starting

    3. Getting Started

  • the tetgen would generate 4 files:
    • example.1.node, example.1.ele, example.1.face, and
      example.1.edge, which are a list of mesh nodes, tetrahedra, boundary faces,
      and boundary edges, respectively

4.2 Command Line Switches

  • -q: add new points to improve the mesh quality
    • maximum radius-edge: default is 2.0 [1.0,4.0]
    • minimum dihedral angle bound: default is 0 [0,90]
  • -r: refine a previously generated mesh
  • -O: mesh optimization level: 0~10 [0,6]
