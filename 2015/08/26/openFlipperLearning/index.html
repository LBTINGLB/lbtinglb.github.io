5.3.1 Introduction

  • To use functions from one of the interfaces you have to:
    • include the interface header file in your plugins header file
    • derive your plugin from the interface class(parent class)
    • add Q_INTERFACES(InterfaceName) macro
    • add the signature ofthe signals and slot

      PS:
