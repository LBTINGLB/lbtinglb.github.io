OpenVim

  • Vi Improved
  • Two modes, insert and normal mode:
    • insert:
      • write text as if in normal text editor
      • press i
    • normal:
      • provide you efficient ways to navigate and manipulate text
      • press Esc
  • Basic movement: h, j, k and l
    • h: move backward
    • j: move to lower line
    • k: move to upper line
    • l: move forward
  • Word movement: w, e, b
    • w: move to the start of next word
    • e: move to the end of the word
    • b: move to beginning of the word
      • press i, start typing at where you are
      • press a, start typing one position after
  • Number powered movement
    • number + movement key
  • Insert text repeatedly
    • number + i + the word you want to repeat
    • press Esc
  • Find a character, f and F
    • number + f + the word you want to find
  • Go to matching parentheses
    • go to ur target parentheses, then press %
  • Go to start/end of a line
    • start line: 0
    • end line: $
  • Find word under cursor, * and #
    • find the next occurrence: *
    • find the previous occurrence: #
  • Goto line, g and G
    • go to the beginning of the file: gg
    • go to the end of the file: G
    • go to specific line: line number + g
  • Search, /text with n and N
    • / + the text you want to search + Enter
    • repeat the search for next and previous occurrences: n and N
  • Insert new line, o and O
    • insert text into new line: o + O
  • Removing a character, x and X
    • x and X delete the character under the cursor and to the left of the cursor, respectively
  • Replacing letter under cursor, r
    • When you need to replace only one character under your cursor, without changing to insert mode, use r.
    • r + the word you want to change
  • Deleting, d
    • d is the delete commandYou can combine it with movement, e.g. dw deletes the first word on the right side of the cursor
    • It also copies the content, so that you can paste it with p to another location (on real Vim).
  • Changing, c
    • c
    • ‘c’hange command deletes the range specified by the next motion command, and also enters insert mode.
  • Append
    • a
  • Repetition with .
  • Visual mode
    • v to choose multiple word
  • Real Vim awaits
    • :w (save)
    • :q (quit)
    • :q! (quit without save)
    • u (undo)
    • Ctrl + R (redo)

Learn Vim Progressively

1. 1st Level - Survive

  • i → Insert mode. Type ESC to return to Normal mode.
  • x→ Delete the char under the cursor
  • :wq → Save and Quit (:w save, :q quit)
  • dd → Delete (and copy) the current line
  • p→ Paste
  • :help <command> -> use :q to exit

2. 2nd Level - Feel comfortable

  • Insert mode variations:
    • a → insert after the cursor
    • o → insert a new line after the current one
    • O → insert a new line before the current one
    • cw → replace from the cursor to the end of the word
  • Basic moves
    • 0 → go to the first column
    • `^`` → go to the first non-blank character of the line
    • $ → go to the end of line
      g`_ → go to the last non-blank character of line
    • /pattern → search for pattern
  • Copy/Paste
    • P -> paste before the current position p is paste after current position
    • yy -> copy the current line,
  • Undo/Redo
    • u -> undo
    • Ctrl-r -> redo
  • Load/Save/Quit/Change File(Buffer)
    • :e → open
    • :w → save
    • :saveas <path/to/file> → save to
    • :x, ZZ or :wq → save and quit (:x only save if necessary)
    • :q! → quit without saving, also: :qa! to quit even if there are hidden buffers.
    • :bn (resp. :bp) → show next (resp. previous) file (buffer)

3. 3rd Level-Better. Stronger. Faster

3.1 Better

  • .->(dot) will repeat the last command.
  • N<command>-> will repeat the commant N times

3.2 Stronger

  • NG -> Go to line N
  • gg -> shortcut gor 1G - go to the start of the file
  • G -> Go to the last line
  • difference between e&E and w&W
    x=(name_1,name_2); # haha
    ||       | |
ew       E W
  • %: Go to the corresponding (,{,[
  • *(resp,#): go to next(resp. previous) occurrence of the word under the cursor)

3.3 Faster

  • `
    • 0y$: go to the beggining of the line, then yank from here, up to the end of the line

4. 4th Level - Vim Superpowers

  • killer features!

4.1 Move on current line

  • 0 -> go to column 0
  • ^ -> go to first character on the line
  • $ -> go to the last column
  • g_ -> go to the last character on the line
  • fa -> go to next occurrence of the letter a on the line. ,(resp) will find the next (resp.previous) occurrence
  • t, -> go to just before the character ,
  • 3fa -> find the 3rd occurrence of a on this line
  • F and f -> like f and t but backward
    x=(name_1,vision3); #this is a comment
    | | i | | | |
    0 ^ fi t) 4fi g     $
  • useful tip: dt"

4.2 Zone selection

  • Only used after an oeprator in visual mode
  • a
  • Suppose the cursor is on the first o of (map (+) ("foo"))
    • vi" → will select foo.
    • va" → will select “foo”.
    • vi) → will select “foo”.
    • va) → will select (“foo”).
    • v2i → will select map (+) (“foo”)
    • v2a) → will select (map (+) (“foo”))

4.3 Select rectangular blocks:

  • Rectangular blocks are very useful for commenting many lines of code. Typically: 0<C-v><C-d>I-- [ESC]

4.4 Completion: and

  • In Insert mode, just type the start of a word, then type < C-p>, maic…

  • J -> join allthe lines together
  • <(resp. >) - indent to the left (resp. to the right).
  • = -> auto indent

  • -> go to desire line -> Do the things you want to do(Uppercase command!) -> Esc

4.7 Splits: split and vsplit

  • :split → create a split (:vsplit create a vertical split)
  • <C-w><dir> : where dir is any of hjkl or ←↓↑→ to change the split.
  • <C-w>_ (resp. <C-w>|) : maximise the size of the split (resp. vertical split)
  • <C-w>+ (resp. <C-w>-) : Grow (resp. shrink) split
  • [copy text into vim]
      1. use Ctrl + C to copy text into system
      1. then in vim editor insert mode, use Ctrl + Shift + V
