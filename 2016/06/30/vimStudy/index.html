  • Why Vim?

    OpenVim

  • Vi Improved
  • Two modes, insert and normal mode:
    • insert:
      • write text as if in normal text editor
      • press i
    • normal:
      • provide you efficient ways to navigate and manipulate text
      • press Esc
  • Basic movement: h, j, k and l
    • h: move backward
    • j: move to lower line
    • k: move to upper line
    • l: move forward
  • Word movement: w, e, b
    • w: move to the start of next word
    • e: move to the end of the word
    • b: move to beginning of the word
      • press i, start typing at where you are
      • press a, start typing one position after
  • Number powered movement
    • number + movement key
  • Insert text repeatedly
    • number + i + the word you want to repeat
    • press Esc
  • Find a character, f and F
    • number + f + the word you want to find
  • Go to matching parentheses
    • go to ur target parentheses, then press %
  • Go to start/end of a line
    • start line: 0
    • end line: $
  • Find word under cursor, * and #
    • find the next occurrence: *
    • find the previous occurrence: #
  • Goto line, g and G
    • go to the beginning of the file: gg
    • go to the end of the file: G
    • go to specific line: line number + g
  • Search, /text with n and N
    • / + the text you want to search + Enter
    • repeat the search for next and previous occurrences: n and N
  • Insert new line, o and O
    • insert text into new line: o + O
  • Removing a character, x and X
    • x and X delete the character under the cursor and to the left of the cursor, respectively
  • Replacing letter under cursor, r
    • When you need to replace only one character under your cursor, without changing to insert mode, use r.
    • r + the word you want to change
  • Deleting, d
    • d is the delete commandYou can combine it with movement, e.g. dw deletes the first word on the right side of the cursor
    • It also copies the content, so that you can paste it with p to another location (on real Vim).
  • Changing, c
    • c
    • ‘c’hange command deletes the range specified by the next motion command, and also enters insert mode.
  • Append
    • a
  • Repetition with .
  • Visual mode
    • v to choose multiple word
  • Real Vim awaits
    • :w (save)
    • :q (quit)
    • :q! (quit without save)
    • u (undo)
    • Ctrl + R (redo)

others

  • u: undo last change
  • Ctrl-R: Redo changes which were undown
  • .: repeat a previous change
  • copy and paste
