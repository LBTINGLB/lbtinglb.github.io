Softwares and packages:

  • ConEmu:
    • force it to become the default cmd
  • Dependency walker:
    • a great tool when ur exe dependency is not working.
  • ninja:
    • a powerful makeFile creater.
    • You need to specify the compiler for it. Open CMake from VS Native Tool Command should be fine.

Tips and ideas:

  • run VTK python wrapper:

  • let ninja choose VS as your C++ compiler

    • open CMD from VS native tools commander
    • cd BUILD_FOLDER
    • cmake-gui SOURCE_FOLDER
  • add a new folder to store all the tools.
  • cout not working for Paraview:
    • check the link in build file, see if LINK_FLAGS = /machine:x64 /debug /INCREMENTAL /subsystem:console or not.
