2.4 Curvature of Surfaces

  • principal curvatures
    • the two principal curvatures at a given point of a surface are the eigenvalues of the shape operator at the point
    • positive when the curve turns inthe same direction as the surface’s chosen normal
  • shape operator
    • tells us how the normal changes as we travel along a direction X
    • the differential df of the Gaussmap f (Weingarten map, dN) is known as the shape operator S
      • df(SX) = dN(X) where X is the tangent vector
  • principal directions S & principal curvatures k & shape operator X
    • S X = k X
  • mean curvature
    • locally describe the curvature of an embeded surface
    • average of the principal curvatures
    • like logical or (is there curvature along at least one direction)
    • zero mean curvature: minimal surfaces (saddle-like)
  • gaussian curvature
    • product of the prinicipal curvatures
    • like logical and(is there curvature along both directions)
      • zero gaussian curvature: developable surfaces
      • nagative gaussian curvature: saddle-like face
      • positive gaussian curvature: bowl-like face

  • hodge duality
    • Suppose that n is the dimensionality of the oriented inner product space and k is an integer such that 0 ≤ k ≤ n, then the Hodge star operator establishes a one-to-one mapping from the space of k-vectors to the space of (n − k)-vectors.

Reading 5 - The Laplacian

  • Laplace operator
    • a differential operator given by the divergence of the gradient of a function on Euclidean space
      • the rate at which the average value of over spheres centered at p, deviate from f(p) as the radius of the sphere grow(在该点p单位元圆的平均值在当圆的半径增加时，函数值f(p)的变化率)
    • denoted as ∇·∇, ∇2, or ∆.
      • the laplacian represents the flux density of the gradient flow of a function.
      • the simplest elliptic operator
  • covariant derivative
    • a generalization of the directional derivative from vector calculus
    • take two inputs:
      • (1) a vector defined at a point p
      • (2) a vector field v,defined in a neighborhood of P
    • output:
      • vector \nabla_{\bold u}{\bold v}(P), also at the point P
      • it’s independent of the manner where it’s expressed in a coordinate system
  • laplace - Beltrami operator
    • the laplace operator in differential geometry
    • appy on surface in Eucidean space as Riemannian and pseudo-Riemannian manifolds
    • the operator can be extended to operate on tensors as the divergence of the covariant derivative

Other definitation

  • injection
    • if every element of the codomain is mapped to by at most one element of the domain. An injective function is an injection
    • map A to B. A has 3 elements, B has 4
  • Bijection
    • a function between the elements of two sets
    • where every element of one set is paired with exactly one element of the other set, and every element of the other set is paired with exactly one element of the first set
    • map A to B. A and B must have same amount of elements.
  • surjective
    • if every element y in Y has a corresponding element x in X such that f(x) = y. The function f may map more than one element of X to the same element of Y.
    • map A to B. A has 4 elements, B has 3
  • homeomorphism
    • a continuous function between topological spaces that has a continuous inverse function. （图形扭曲变换）
    • key idea: homeomorphism preserve everything about topology.
  • isomorphism
    • a homomorphism that admits an inverse
    • the canonical map from a finite-dimensional vector space V to its second dual space is a canonical isomorphism; on the other hand, V is isomorphic to its dual space but not canonically in general.
  • quotient space)
    • the result of identifying or “gluing together” certain
  • homomorphism
    • in abstract algebra, it’s a structure-preserving map between two algebraic strucutres.(rings or vector spaces)
  • quotient space)
    • which is the result of identifying or “gluing together” certain points of a given topological space.
  • topological space
    • given a set X, a topology T on X is a collection of subsets of X with the following properties:
      • the empty set and X are both in T
      • The union of any collection of open sets is an open set.
      • The intersection of any finite open sets is an open set
        • If A,B $\subset$ T, then A$\cap$B belongs to T
      • every element of T as a T-open set
    • Trivial Topology:
      • `smallest or coarsest possible topology: T = {X, $\emptyset$}
    • Discrete Topology:
      • Largest or finest possible topology: T = power set.
  • continuous map
    • a function for which sufficiently small changes in the input result in arbitarily small changes in the output. Otherwise, a function is said to be a discontinuous function.
  • compactness: A topological space is compact if every open cover has a finite subcover.
  • Closure & Interior & Boundary
    • Let A be a subset of a topologial space X
      • closure cl(A): the smallest closed set containing A
      • interior int (A): the largest open set contained by A
      • boundary $\partial A$: the intersetion of cl(A) and cl(X\A)
  • Riemannian manifold
    • 一个微分流形，其中每点p的切空间都定义了点积，而且其数值随p平滑地改变。它容许我们定义弧线长度，角度，面积，体积，曲率，函数梯度及向量域的散度。
