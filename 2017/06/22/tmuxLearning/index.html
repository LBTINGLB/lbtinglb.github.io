A Quick and Easy Guide to tmux

  • window managment + session managment
  • trigger commands: prefix key + command key
    • C-b %: spit window vertically
    • C-b ": spit window horizontally
    • C-b <arrow key>: switch between panes
    • C-d/type inexit: close panes
    • C-b c: create windows
      • C-b p: switch to previous window
      • C-b n: switch to next window
      • C-b <number>: switch to specific window by number
  • detach retach session
    • C-b d: detach current session
    • C-b D: detach a specific session
    • tmux ls: list all available sessions
    • tmux attach -t <index of session>
    • tmux new -s database: create a session with desired name
    • tmux rename-session -t 0 database: rename a session
    • C-b ?: get list of available commands
    • C-b z: make a pane go full screen. Hit C-b z again to shrink it back to its previous size
