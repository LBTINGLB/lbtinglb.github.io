Simple Tutorial with OpenMP with concrete examples

  • #pragma omp parallel num_threads(3): explicitly specify the number of threads in OpenMP
  • omp_get_thread_num and omp_get_num_threads: get number of threads and current thread id
  • private variables: std::thread_local
  • #pragma omp critical: ensures only 1 thread enters the block at a time

OpenMP tutorial
