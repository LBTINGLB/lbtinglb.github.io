Week3: Greedy algorithm

  • sample problem: largest number, car fueling, etc
  • greedy algorithm:
    • make some greedy choice, reduce it to a smaller problem; iterate
  • a greedy choice as safe move: there is an optimal solution consisitent with the first move

    Grouping Children problem

Celebration Party Problem

  • problem def: organzie the children into minimum possible number of groups so that the age of any two children in the same group differ by at most one year.
  • reformulate the problem into a math problem -> find a safe move -> write the algorithm
  • strict weak ordering
    • in C++ std::sort function, it requires strick weak ordering. So you can not return true if a==b in your compare function.

Week4: Divide-and-Conquer
